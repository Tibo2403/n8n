# Portfolio Brief

## Problem

Automation projects need readable workflow exports, validation, and safe examples that can be reviewed without exposing credentials.

## What This Shows

- n8n workflow design for social publishing and notification flows.
- JSON export hygiene and CI validation.
- Operational documentation for importing, exporting, and reviewing automation.
- Credential-safe examples for demos.

## Demo Flow

1. Run `node scripts/validate-workflows.mjs`.
2. Import `workflows/auto-social-publisher.json` into n8n.
3. Review the deterministic demo post output.
4. Import additional examples for digest and approval workflows.

## Recruiter Notes

This project demonstrates automation thinking, operational discipline, and safe handling of workflow exports.
